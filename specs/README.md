### Core Systems

**Physics**: Rapier.js is the source of truth. Three.js renders this state with delta time.

**Gameplay**: Mobile phones act as controllers, host runs on Chrome, server handles socket passthrough.

**Assets**: Music generated by Suno AI. 3D models can be imported. Cars, maps, and physics defined in JSON files.
